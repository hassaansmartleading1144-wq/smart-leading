# Page previews (no WordPress required)

Browser previews for theme page templates.

## Digital Marketing page

**Local preview** (from repository root):

```bash
python3 -m http.server 8765
```

Then open:

- http://localhost:8765/preview/
- http://localhost:8765/preview/digital-marketing-preview.html

**GitHub Pages** (after the `Preview Pages` workflow runs and Pages is enabled):

https://hassaansmartleading1144-wq.github.io/smart-leading/preview/

## Notes

- Uses the same CSS/JS as the WordPress template (`digital-marketing-template.php`).
- On the live site: **Pages → Add New → Template: Digital Marketing → Publish**.
